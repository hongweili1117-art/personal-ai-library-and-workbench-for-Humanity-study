个人 AI 智库 · 极简使用说明
========================================

这是一个跑在你自己电脑上的 AI 学术阅读工具。你的 PDF、笔记、分析都只存在
本机的 data 文件夹里，不会上传到任何服务器。


【第一步：安装（只需一次）】

  Windows：
    双击  一键安装_Windows.bat

  苹果 Mac：
    双击  一键安装_Mac.command
    （如果提示"无法打开"，在文件上点右键 → 选"打开"）

  Linux：
    打开终端，输入：  bash start_linux.sh

  安装时会自动下载一些组件，第一次可能要几分钟，请耐心等、别关窗口。
  ——如果提示"没有 Python"，按它给的网址装一下 Python 再重来。
    （Windows 装 Python 时记得勾选 "Add Python to PATH"）


【第二步：以后每次启动】

  Windows：双击桌面上的"个人AI智库"，或文件夹里的  启动_Windows_无黑框.vbs
  Mac：    双击  启动_Mac.command
  Linux：  bash start_linux.sh

  启动后会弹出「个人 AI 智库」程序窗口（装了 pywebview 时）；
  若没有该组件，则用系统默认浏览器打开，并附带一个very小的「运行中 / 退出」窗口。
  想关掉程序：关闭程序窗口即可（用浏览器方式时，点那个小窗里的"退出程序"）。
  —— 全程不再有黑色命令行窗口；也不会再因为关掉黑框而出现"fail to fetch"。


【第三步：填一次 AI 钥匙】

  点页面右上角的设置，选一个 AI 引擎并填入它的 API Key：
    · 用 DeepSeek / OpenAI / Z.ai → 填对应的 Key
    · 用 Claude → 填 Anthropic 的 Key
  （"免费"档只能翻译，其它 AI 功能都需要上面任选一个引擎）

  钥匙只存在你自己电脑上，不会外传。


【怎么用】

  左边上传 PDF，上方一排按钮：
    问答 / 文献卡片 / 摘要 / 批判与思考 / AI 智库 / 对比 / 图表 / 笔记 / 原文

  · 想让 AI 当"严厉的同行"挑文章毛病 → 点"批判与思考"
  · 想就一个观点让 AI 翻十几篇文献找证据、再导出报告 → 点"AI 智库"
  · 扫描件看不了字 → 打开"原文"点 OCR


【把程序打包发给朋友（免安装，朋友无需等待安装）】

  如果你想把程序发给朋友、又不想让对方等"一键安装"下载组件，可以先在
  你自己电脑上打包成"免安装版"——把 Python 和所有组件一起打进去，
  对方拿到后无需装 Python、无需联网，直接双击就能用。

  Windows：双击  一键打包_Windows.bat
  苹果 Mac：双击  一键打包_Mac.command

  打包第一次要几分钟。完成后，本文件夹里会出现一个压缩包：
      个人AI智库_可直接运行_Windows.zip   （或 _macOS.zip）
  把这个压缩包发给朋友即可。对方解压后：
      · Windows：双击里面的  个人AI智库.exe
      · Mac：    双击里面的  双击启动.command（首次提示无法打开就右键→打开）

  注意：
    · 这个压缩包体积较大（一两百 MB 很正常，因为连运行环境都打包进去了）。
    · 打包结果只适用于"你当前的系统"：在 Windows 上打包的发给用 Windows 的朋友，
      在 Mac 上打包的发给用 Mac 的朋友。要兼顾两种系统，就在两种系统上各打包一次。
    · 打包前请确认你这边已经能正常运行本程序（即已完成过一次安装）。


【遇到问题】

  · 浏览器没自动开：手动在浏览器输入  http://127.0.0.1:5000
  · PDF 打不开 / 提示"打开失败：Failed to fetch"：
      本程序已内置离线 PDF 引擎，就在文件夹里的 lib\ 目录
      （pdf.min.js 与 pdf.worker.min.js）。请勿删除 lib 文件夹；
      若不慎删了，把这两个文件放回 lib\ 再刷新页面即可。
  · 双击"启动_Windows.bat"窗口一闪而过：现在脚本出错会停下并显示原因，
      若仍一闪而过，请在该文件上点右键 →"以管理员身份运行"或"运行"。
  · Mac 双击 .command 提示"无法打开"：在文件上点右键 →"打开"；
      或打开"终端"，把文件夹拖进去后执行一次：chmod +x *.command
  · PDF 打开后报"PDF 文件已不在原位置"：该 PDF 被移动或删除了，在书架重新导入即可。
  · 程序闪退/报错：看 data 文件夹里的 logs，把里面的错误发给分享给你这个程序的人
  · 安装下载很慢：让对方教你用国内镜像（pip 加 -i 参数）

  安全相关见 SECURITY.md。


【本版新增 / 改动】

  1. 修复「粘贴网页链接→转干净 PDF」：以前会生成空白、或提示要装 trafilatura。
     现在改用内置方式抓取与排版，无需再装任何额外组件，正文也更干净。
  2. 文件可右键管理 + 二级书架（文件夹）：
     · 在书架里【右键】任意文件 → 打开 / 移动到分组 / 删除（删除会一并清掉它的
       PDF、相关高亮、笔记与索引；也可点文件上的 📁 按钮，里面同样有"删除此文件"）。
     · 用 📁 把文件归入分组，分组会折叠成一层文件夹；【右键分组标题】可重命名或解散分组
       （解散只是把文件移到"未分类"，不会删文件）。
  3. 新增更多 AI 引擎：在「设置 → AI 引擎」里新增了 通义千问 Qwen（含 qwen3.7-plus）
     和 Kimi（月之暗面）；智谱 GLM 仍含 5.2 / 5.1 等。各引擎都能填自己的 Key 并随时切换。
  4. 框选 OCR 升级：框选区域后可选四种任务——提取文字 / 翻译成中文 / 解读图表 /
     公式转 LaTeX，并支持用 通义千问 / 智谱GLM / Kimi / OpenAI / Claude 的视觉模型识别
     （选了这些引擎时会自动用其视觉版，如 qwen3-vl-plus、glm-4.6v）。
  5. 附带一个命令行小工具「把扫描版PDF转成可搜索PDF.py」：把整本扫描版 PDF 转成
     可选字/可搜索的 PDF（本地、免费、不联网；需另装 Tesseract + Ghostscript，
     详见该脚本顶部说明）。日常框选识别用上面第 4 点即可，无需这个工具。
